Skill Forge

An autonomous experiment loop that makes AI skills better while you sleep. Point it at any skill, wake up to measurable improvements.

View on GitHub Quick Start
Analyze → Hypothesize → Mutate → Evaluate → Score → Keep / Revert ↺

Three Agents, One Loop

Each experiment cycle is driven by three specialized agents working in sequence.

🔬

Hypothesis Agent

The Scientist

Analyzes grading results and failure patterns. Produces a testable hypothesis with a concrete mutation proposal targeting the root cause.

🩹

Mutator Agent

The Surgeon

Applies one minimal, focused change to SKILL.md. Documents exactly what changed and why, keeping mutations atomic and reversible.

⚖

Scorer Agent

The Judge

LLM-as-Judge evaluating output quality across four dimensions: task completion, quality, robustness, and efficiency.

Objective Scoring

Every mutation is measured against a composite metric. No vibes, no gut feeling — just numbers.

Composite Score
assertion_pass_rate × 0.80  +  efficiency_score × 0.20
With LLM-as-Judge (Blind A/B)
assertions × 0.50  +  llm_judge × 0.30  +  efficiency × 0.20

Built to be Robust

Production-grade safeguards so you can run it unattended overnight.

Train/Test Split

60/40 split prevents overfitting to specific eval cases.

Crash Recovery

Classifies errors (script bug, infra, eval) and continues the loop.

Session Tagging

Unique tags per run prevent conflicts from parallel experiments.

Transcript Logging

Full experiment transcripts for debugging and reproducibility.

Eval Rotation

Fresh eval queries after 5 experiments to maintain diversity.

Generalization Check

Hypothesis agent must explain why each change generalizes.
